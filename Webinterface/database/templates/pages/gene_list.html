Information:

Motif scanning means finding all known motifs that occur in a sequence. This web-tool allows you to display all known motifs that occur within a sequence. If snp-variant has been detected within a motif, it will also be displayed

How To:

In order to display the motifs that occur in a sequence, there are two steps necessary.
First choose the gene
Second step is to input the sequence and then choose a platform. Different platforms has been used to analyze the gene and to find motifs. Additionally, it is possible to display common motifs between two platforms. Required steps are an input sequence, and to select two platform.
